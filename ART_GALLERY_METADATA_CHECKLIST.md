## ‚úÖ REQUIRED: Artwork Details

Each artwork submission must include:

### 1. Basic Information

- [ ] **Artwork Title** (e.g., "Urban Sunset," "Digital Dreams," "Ocean Whispers")
- [ ] **Medium** (select from platform options):
  - Oil Painting | Acrylic Painting | Watercolor | Gouache

  - Sculpture (Bronze) | Sculpture (Stone) | Sculpture (Wood) | Sculpture (Ceramic) | Sculpture (Mixed Media)

  - Mixed Media | Collage | Textile Art | Installation Art (Documentation)
  - Other (specify in description)

- [ ] **Year Created** (YYYY format, e.g., 2025)
- [ ] **Dimensions**
  - For 2D works: Width x Height (e.g., "60cm x 80cm" or "24in x 36in")
  - For 3D works: Width x Height x Depth (e.g., "30cm x 40cm x 15cm")
  - For digital works: Pixel dimensions and file format (e.g., "4000px x 6000px, PNG")

### 2. Edition Information

- [ ] **Edition Type** (select one):
  - **Original (One-of-a-kind):** Unique artwork, no reproductions exist
  - **Limited Edition:** Specify total number and current number (e.g., "5/50" ‚Äî this is the 5th print of 50 total)
  - **Open Edition:** Unlimited reproductions available (prints, digital downloads)
  - **Artist Proof:** Special edition outside main run, typically marked "AP 1/10" or similar

- [ ] **Edition Details** (if Limited Edition or Artist Proof):
  - Total edition size (e.g., 50 prints)
  - Current edition number (e.g., this is #5)
  - Signature details (signed, numbered, dated)

### 3. Description & Context

- [ ] **Artwork Description** (100-500 words):
  - **Concept:** What is the artwork about? What themes or ideas does it explore?
  - **Inspiration:** What inspired you to create this piece?
  - **Technique:** What methods, tools, or processes did you use?
  - **Materials:** What materials are used? (e.g., oil on canvas, archival inkjet on paper, bronze casting)
  - **Context:** Any relevant background (series, exhibition history, awards)

**Example:**  
"Urban Sunset is a vibrant exploration of urban landscapes at dusk, capturing the interplay between natural and artificial light. Inspired by my daily commute through London, this piece celebrates the beauty found in everyday city moments. Created using oil on canvas with layered glazing techniques, the painting builds depth through multiple transparent layers. This work is part of my 'City Series' collection, which has been exhibited at the Saatchi Gallery and shortlisted for the Royal Academy Summer Exhibition."

### 4. Provenance & Authenticity

- [ ] **Condition Report** (required for all artwork):
  - Describe the current physical state of the artwork
  - Note any imperfections, repairs, wear, or damage
  - Be honest and transparent (protects both artist and buyer)
  - Examples:
    - "Excellent condition. No tears, stains, or repairs. Colors remain vibrant."
    - "Good condition with minor frame wear on bottom edge (photograph included)."
    - "Fair condition. Small repair to upper left corner (professionally restored 2023)."

- [ ] **Artist Declaration** (required ‚Äî must confirm all statements):
  - "I am the original creator of this artwork."
  - "I hold all copyright and intellectual property rights."
  - "This artwork does not infringe any third-party rights."
  - "All provided information is accurate and complete."
  - "This artwork does not fall under excluded categories (mass-produced, purely AI-generated, reproduction, functional object, offensive/unlawful content, or unverified)."

- [ ] **Authenticity Declaration:**
  - Signed statement confirming originality
  - Optional: Upload certificate of authenticity (if already issued)
  - Platform will issue official certificate upon approval

- [ ] **Provenance / Ownership History** (required if applicable):
  - List previous owners, if any
  - Include dates of ownership
  - Note if commissioned work (include commissioner name with permission)
  - Example: "Originally commissioned by [Client Name] in 2020. Owned by artist since completion."

- [ ] **Exhibition History** (optional but recommended for auction):
  - List galleries, museums, or exhibitions where artwork has been displayed
  - Include dates and locations (e.g., "Saatchi Gallery, London, June 2024")
  - Strengthens provenance and increases buyer confidence

- [ ] **Awards or Recognition** (optional):
  - List any awards, prizes, or critical acclaim received
  - Include press mentions, reviews, or publications featuring the artwork

## üîÑ How It Works in Practice (4-Week Auction Cycle)

**1. Artist Completes Metadata Form**
- All required information gathered using this checklist
- Artwork images uploaded (high-resolution primary + optional detail shots)

**3. Auction Runs for 4 Weeks (Fixed Duration)**

- Provided reserve/minimum price is met
