A Streamlit application for Formula 1 data analysis and visualization.

## Purpose

GETS Studio F1-V2 is designed to provide insights and analytics for Formula 1 racing data, enabling users to explore race statistics, driver performance, and team standings.

## How to run it on your own machine

1. Install the requirements

   ```bash
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```bash
   $ streamlit run streamlit_app.py
   ```

## Features

- Interactive data visualization
- Real-time race analysis
- Historical performance tracking

## Tech Stack

- **Streamlit** - Web application framework
- **Pandas** - Data manipulation and analysis
