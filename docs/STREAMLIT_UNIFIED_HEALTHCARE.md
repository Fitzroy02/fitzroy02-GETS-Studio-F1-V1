# Streamlit Unified Healthcare Dashboard

## Features
- **Doctor Dashboard:** View/edit patient records, schedule meetings, manage shared files.
- **Patient Dashboard:** View own records, confirm appointments, access shared files.
- **Co-worker Dashboard:** Fill reports, join meetings, collaborate on shared files.
- **Audit Trail:** Every action logged in JSON for transparency.

## Usage
1. Place `dashboard_data.json` in the project root.
2. Run `streamlit run app.py`.
3. Use the sidebar to switch between Doctor, Patient, and Co-worker dashboards.
4. Interact with records, meetings, and files according to role permissions.
