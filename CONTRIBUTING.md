# Contributor Pledge

By contributing to this project, I pledge to uphold the principles of ecological stewardship, transparent governance, and strict data ethics. I will:

- Protect the dignity and privacy of all users, ensuring **no child data** is ever monetized or shared.
- Work only with **adult user statistics**, and only with explicit consent, transparency, and auditability.
- Anchor every contribution to visible ecological impact, such as sapling planting and proportional rewards.
- Respect role-based permissions for parents, practitioners, and administrators, ensuring shared governance.
- Write clear, copy-friendly documentation and code that makes stewardship accessible to all.

I understand that violations of these principles undermine the trust and mission of this project, and I commit to contributing responsibly, ethically, and transparently.
